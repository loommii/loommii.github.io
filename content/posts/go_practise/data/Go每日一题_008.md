如何确认两个 map 是否相等？

{{< togglecontent label="🔑 答案解析：" >}}

map 深度相等的条件：

都为 nil
非空、长度相等，指向同一个 map 实体对象
相应的 key 指向的 value “深度”相等

直接将使用 `map1 == map2` 是`错误`的。这种写法只能比较 map 是否为 nil。

```go
package main

import "fmt"

func main() {
    var m map[string]int
    var n map[string]int

    fmt.Println(m == nil)
    fmt.Println(n == nil)

    // 不能通过编译
    //fmt.Println(m == n)
}
```

方案一: 遍历（比较麻烦）
遍历 map 的每个元素，比较元素是否都是深度相等。
方案二: 反射

```go
package main

import( 
    "fmt"
    "relflect"
)
func main() {
    var m map[string]int
    var n map[string]int

    fmt.Println(reflect.DeepEqual(m,n))
}
```
